# 🏥 医疗处方安全风控 AI 扫描器 (Prototype)

## 🌟 项目概述
这是一个针对互联网医疗场景设计的 **AI 决策支持系统 (CDSS) 原型**。
旨在解决 40+ 高龄患者在肾功能不全情况下，由于人为疏忽导致的“二甲双胍”禁忌症用药风险。

## 🧠 核心逻辑架构 (Solution Logic)
作为解决方案经理，我为本系统设计了以下核心逻辑：
1. **原子性拦截**：当 eGFR < 30 时，系统执行 Hard Block（硬拦截），禁止处方进入数据库。
2. **容错性设计**：
   - 绝对禁忌 (eGFR < 30)：强制拦截 + 逻辑解释。
   - 相对风险 (30 <= eGFR < 45)：柔性预警 + 减量建议。
3. **闭环管理**：异常行为自动记录日志，并同步推送至药剂师工作站。

- **逻辑协议**: 结构化提示词工程 (Structured Prompt Engineering)
